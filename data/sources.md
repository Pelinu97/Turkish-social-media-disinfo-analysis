## Open-Source Data & References
All sources used in this project are publicly available, secondary-source materials. No classified, proprietary, or non-public information has been used, and no original social media scraping or account-level data collection was conducted for this project — analyses in /analysis synthesize findings already published by the organizations listed below.


## Social Media & Public-Record Platforms
Twitter/X Advanced Search — used only to locate publicly available, already-reported posts referenced in cited case studies, not for original data collection
Telegram — referenced as a platform discussed in source reporting; no original channel monitoring was conducted for this project
Internet Archive — used for accessing archived versions of pages cited in source reporting

Note: Tools like TweetDeck/X's real-time monitoring features are listed in some OSINT workflows but were not used as part of this project, since no original monitoring was performed. They are omitted here to keep this list accurate to what was actually done.

## Methodological Note

This list reflects sources actually consulted or cited in /analysis. It does not include tools or platforms that would be needed for an original, dataset-level OSINT investigation (e.g., systematic network-graph construction from a defined account sample) — that would be a logical next phase of this project, distinct from the source-synthesis work documented here.
