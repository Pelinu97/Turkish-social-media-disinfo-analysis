# Visuals
This folder is reserved for visual products supporting the analyses in /analysis. No original social media data collection underlies these visuals — each planned visual is sourced from the case studies and reporting cited in the analysis documents, not from independent scraping.

## Planned Content
Narrative theme map — visual grouping of the four documented narrative themes (Analysis 1) with source attribution per theme
Case study timeline (2022–2024) — placing the four case studies (Analysis 2) against major Russia-Ukraine war milestones, to make clear which narrative spikes are documented versus inferred from proximity to events
Platform distribution chart — summarizing which platforms each cited case study covers, to make source-coverage gaps visible (e.g., Telegram is consistently under-covered relative to Twitter/X and Facebook in available reporting)

## Status
These visuals are planned, not yet produced. Each, when built, will carry a per-element source citation so that any single data point in the visual can be traced back to a specific case study or report rather than presented as an independent finding.
