# Turkish Social Media Disinformation Analysis

**Open-Source Intelligence (OSINT) Analysis of Pro-Russian Disinformation Narratives Targeting Turkish Social Media (2022–2024)**

---

## 📌 About This Project

This project is an open-source intelligence analysis examining pro-Russian disinformation campaigns targeting Turkish social media platforms, with a focus on narratives related to the Russia-Ukraine war, NATO membership, and Turkish foreign policy.

## 🎯 Objectives

- Identify and document pro-Russian disinformation narratives circulating on Turkish social media
- Analyze the methods and networks used to spread coordinated inauthentic behavior
- Assess the potential impact on Turkish public opinion and foreign policy

## 🗂️ Repository Structure

- `/data` — Open-source datasets and references
- `/analysis` — Intelligence assessments and written reports
- `/visuals` — Network maps, charts, and infographics

## 🔍 Methodology

All research is based exclusively on open-source information including social media archives, fact-checking organizations, academic research, and investigative journalism.

## ⚠️ Disclaimer

This project is purely academic and based entirely on publicly available information.
